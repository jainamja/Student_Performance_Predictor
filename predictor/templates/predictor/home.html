About This Project

The Student Performance Predictor is a Machine Learning based web application developed using Django, Python, and Scikit-learn.

It predicts student academic performance based on factors like:

  • Study Hours
  • Attendance
  • Previous Scores
  • Motivation Level
  • Family Support

This project also includes:

  • Authentication System 🔐
  • Analytics Dashboard 📊
  • Prediction History 🗂️
  • Interactive Charts 📈
